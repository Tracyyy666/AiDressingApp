#  AI 穿搭灵魂助手

> 基于智谱大模型（GLM-4-flash / GLM-4V）的智能穿搭推荐系统  
> 支持性别、体型、季节、图片肤色分析，告别同质化搭配

---

##  项目简介

**AI 穿搭灵魂助手** 是一款利用大语言模型和视觉模型为个人提供个性化穿搭建议的 Web 应用。用户只需输入身高、体重（根据用户心理底层逻辑设计了滚轮而非直接手动输入自己的真实体重）、性别、体型、偏好风格、当前季节（春/夏/秋/冬），并可上传照片进行肤色分析，AI 即可生成包含上装、下装、鞋的完整搭配方案及详细穿搭逻辑说明。界面设计为暖色调，高贵不失典雅，符合当代现代审美。对不知道该怎么搭配，不懂得搭配技巧的时尚小白非常友好。

---

##  核心功能

| 功能 | 说明 |
|------|------|
|  **性别选择** | 支持女性 / 男性，AI 会针对不同性别推荐适配服饰 |
|  **身高 / 体重** | 滑块选择身高，滚轮/输入体重 |
|  **体型选择** | 梨型、苹果型、沙漏型、矩形、倒三角，或由 AI 分析 |
|  **季节适配** | 春、夏、秋、冬，AI 根据季节推荐材质、厚度与色彩 |
|  **风格偏好** | 休闲、通勤、复古，支持自定义风格输入 |
|  **上传照片** | 可选上传人物照片，AI 分析肤色调性（冷暖）和面部氛围，融入色彩建议 |
|  **AI 推荐** | 返回上装、下装、鞋履的详细描述 + 搭配逻辑说明 |
|  **响应式界面** | 适配 PC / 手机，优雅的复古时尚风格 |

---

## 🛠️ 技术栈

- **后端框架**：Flask (Python)
- **AI 模型**：智谱 AI（GLM-4-flash 文本生成、GLM-4V 多模态图像分析）
- **数据库**：SQLite（存储历史记录）
- **前端**：HTML5 + TailwindCSS + 原生 JavaScript
- **版本控制**：Git + GitHub
- **环境管理**：python-dotenv
